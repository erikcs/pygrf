# pygrf

#### Installation
(requires: `numpy`, `cython`, C++ compiler)

```bash
pip install git+https://github.com/erikcs/pygrf.git
```

or

```bash
git clone https://github.com/erikcs/pygrf.git
pip install -e .
```
